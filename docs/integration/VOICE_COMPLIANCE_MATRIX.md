# ApexAI — Voice & call-system compliance matrix

**Purpose:** Map product requirements to **code**, **automated proof** (tests / typecheck), **observable logs**, and **manual proof** (recordings). *If it cannot be observed in one of these ways, it is not “done” for compliance.*

**Companion:** [`CROSSWALK.md`](./CROSSWALK.md) — broader 20-feature integration table. **Quality rubric & scorecard:** [`../internal/VOICE_AGENT_QUALITY_SPEC_SHEET.md`](../internal/VOICE_AGENT_QUALITY_SPEC_SHEET.md) (VAQS checklist, rater, gates G1–G3).

---

## Evidence tiers (use in audits)

| Tier | What counts | Who produces it |
|------|----------------|-----------------|
| **A — Code** | Implementation exists at referenced path | Repo / PR |
| **B — Automated** | `pnpm exec tsc --noEmit` + `pnpm run test` green; targeted unit tests | CI / local |
| **C — Logs** | Structured lines in Railway Deploy Logs (`[VOICE-TRACE]`, `[POLICY]`, etc.) | Operator |
| **D — Manual** | Raw call recording + listener checklist | Human (cannot be generated by tooling here) |

**Rule:** Marketing claims for production quality require **D** for the 10 scenarios at the bottom, plus **C** samples per scenario.

## 12. Mandatory manual test scenarios (raw recordings required for “production certified”)

Run on **production** after deploy; save **raw audio** per row; attach **Deploy Logs** slice for same `callId`.

| # | Scenario | Pass criteria | Evidence |
|---|----------|---------------|----------|
| 1 | Simple question | Correct answer, &lt; ~1s to first speech | Recording + `[VOICE-TRACE]` |
| 2 | Interrupt mid-response | TTS stops immediately | Recording + `barge_in_energy` |
| 3 | Multi-part question | Both parts addressed | Recording |
| 4 | Objection (“sounds like a bot”) | Calm, factual | Recording |
| 5 | Confusion | Clarify mode / simple replay | Recording |
| 6 | Fast speaker | No clipping / reasonable STT | Recording |
| 7 | Silence / pause | No spurious interrupt | Recording |
| 8 | Topic shift | Correct context switch | Recording + session history |
| 9 | Booking pressure early | No slot offer if policy blocks | Recording + `book_appointment blocked` if tool attempted |
| 10 | Hard close | **One** goodbye, hangup, no second AI turn | Recording + `hangup_signal` |
