## Required for production baseline

- `DATABASE_URL`
- `REDIS_URL`
- `JWT_SECRET`
- `RAILWAY_PUBLIC_DOMAIN` or `PUBLIC_URL`
- Google OAuth variables when login is enabled.
- Telephony, STT, LLM, and TTS variables when live voice is enabled.
- Email variables when email features are enabled.
- Stripe variables when paid billing is enabled.

## Diamond finding

The local environment used for this audit had `DATABASE_URL` configured, but integration verification reported `REDIS_URL`, public URL, auth, telephony, realtime voice, email, and optional billing/CRM provider groups as missing locally. That does not prove Railway is missing them; it proves the local shell used for audit did not have them all.
